## Z-Tool backend

* Time
* Page view count for <http://www.jack003.com>
* Parse rss from rss URL
* Request/IP infomation
* Domain Name Information
* DNS Record

### Usage (ubuntu 14.04)

1. change vps domain name (IP address) at `fibfile.py`
2. change `domain_name` to your domain name
3. run `fab install`
4. run `fab deploy`
